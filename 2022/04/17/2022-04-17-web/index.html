用 Github page + Hexo 架設網頁

發表於 2022-04-17 更新於 2022-06-04 分類於 自學

原本搞了很久成功架好網頁，結果硬碟突然壞掉重灌電腦資料都不見了，於是決定寫一篇架網頁的文，來記錄架網站的過程以及整理我遇過的問題，希望能幫助到其他人。喔然後我第二次架設時又架了好幾個小時QQ。

前置

你需要:

  • VScode
  • Node.js
  • Git
  • 在 Github 創一個 用戶名.github.io 的儲存庫
    ex : 我的名字是 icerain0930 就創 icerain0930.github.io

安裝 Hexo

用 CMD 打
npm install -g hexo-cli
創一個叫 的資料夾(會把 hexo 的東西放進去)
hexo init <name>
用 VScode 打開你所創建的資料夾後打開終端機輸入
git clone https://github.com/用戶名/用戶名.github.io.git

安裝 Next

git clone https://github.com/theme-next/hexo-theme-next themes/next
然後可以在 Next/_config.yml 去更改設定
記得要在原本的 _config.yml 找到 theme 改成theme:Next

更改 Next (不用的用 ‘#’ 來註解)

改變版面(我用Gemini)

設定部落格

讓 hexo 可以佈署到 github 上
複製 github 的位置後貼到 _config.yml 裡的 repo 後面

接著在終端機打

接著使用
hexo d
應該要跳出登入 Github 的畫面，如果跳出

就改雙引號裡的東西，之後再佈署一次

新增標籤、分類、歸檔頁面

之後在 source 下找到你增加的頁面後點進 index.md

增加type: tags (其他同理)

寫文章!

每次更動後在終端機打

來佈署到網站上

創一個新的文章

hexo new 樣式 名稱
樣式為 scaffolds 下的 draft , page , post (預設)
ex: hexo new post owo
可以自行更改樣式

讓文章顯示”閱讀更多”

在文章中加入

來自行決定要斷開的地方

# web
